# USB Print Pro

App Android para imprimir documentos, imagenes y texto desde tu dispositivo
movil mediante un cable USB con adaptador OTG.

## Caracteristicas

- Deteccion automatica de impresoras USB (clase 7)
- Impresion de texto con formato PCL
- Soporte para PDF, imagenes y HTML (proximamente)
- Modo prueba sin necesidad de cable OTG

## Requisitos

- Android 8.0 (API 26) o superior
- Cable USB OTG
- Impresora con soporte PCL (HP, Brother, Canon, etc.)
